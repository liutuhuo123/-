# 周易起卦程序

一个基于Python的周易起卦程序，用于占卦周易，提供简单的图形界面。

## 功能特点

- 支持传统周易起卦算法
- 提供简单直观的图形界面
- 无需网络连接即可运行
- 输出详细的卦象解释

## 安装要求

- Python 3.x
- tkinter (Python标准库，无需额外安装)

## 使用方法

1. 确保已安装Python 3.x
2. 运行图形界面程序：
   ```
   python zhouyi_gui.py
   ```
3. 点击"起卦"按钮获取卦象
4. 查看卦象解释和建议
